## 📖 COMPREHENSIVE DOSSIER

### Current roles

### Education & career
- **PhD, Harvard, 2014**; Gates Scholar at Cambridge; Simons Fellow at UC Berkeley; ML researcher at Microsoft Research New England.
- Joined Stanford in 2016; leads ~15 PhD students/postdocs. At least four companies have spun out of his group.

### Areas of expertise
ML foundations (Data Shapley data valuation); AI for medicine & genomics (FDA-cleared EchoNet; pathology foundation models; Trial Pathfinder); trustworthy/efficient AI; and AI agents / multi-agent systems.

### Signature frameworks
**1) TextGrad — "automatic differentiation via text."** Treats a compound AI system (chained LLM calls + tools + code) as a computational graph and backpropagates *textual* feedback (natural-language "gradients") to improve prompts, code, molecules, even treatment plans. PyTorch-like API. Results: GPQA 51%→55% (GPT-4o zero-shot); +20% relative on LeetCode Hard. (Yuksekgonul, Bianchi, Boen, Liu, Huang, Guestrin, Zou; arXiv Jun 2024; "2024 Best AI Paper.")

**2) The Virtual Lab.** An LLM Principal Investigator (PI) agent coordinates specialized LLM scientist agents (e.g., immunologist, computational biologist, ML engineer) via structured "research meetings"; a human gives only high-level steering. Nanobody case (Nature, Jul 29, 2025; Swanson, Wu, Bulaong, Pak, Zou): built a pipeline using ESM + AlphaFold-Multimer + Rosetta → 92 candidate nanobodies → 2 with improved binding to JN.1/KP.3 variants while retaining ancestral binding. Among the first AI-agent teams to produce experimentally validated therapeutic candidates.

**3) Paper2Agent.** "Reimagining Research Papers As Interactive and Reliable AI Agents" (arXiv 2509.06917, Sep 2025; GitHub jmiao24/Paper2Agent). A multi-agent system that automatically turns a paper + its codebase into an interactive AI agent: it analyzes the paper, builds a Model Context Protocol (MCP) server, then iteratively generates and runs tests to make the tools reliable and reproducible. Users run complex analyses via natural language; each tool is validated against the paper's reported results. Example: generated 22 AlphaGenome MCP tools in ~3 hours on a laptop, no human intervention. Turns passive research into active, reusable agents.

**4) Other notable work.** Data Shapley (equitable data valuation; ~1,562 cites); pathology vision-language foundation model (~1,011 cites); SyntheMol (generative antibiotic design — ~25,000 candidates in <9h; 6 killed resistant *A. baumannii*; NYT 2024 "Good Tech"); EchoNet (FDA-cleared cardiac AI); Gradio (open-source ML demos, 1M+ developers); Trial Pathfinder (more-inclusive clinical-trial eligibility).
